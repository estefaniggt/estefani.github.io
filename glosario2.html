FUNCIONES ARRAY

every()


El método every () comprueba si todos los elementos de una matriz pasan una prueba (proporcionada como función).



Presione para comprobar si cada elemento de la matriz tiene un valor de 20 o más.


fill()


El método fill () llena los elementos especificados en una matriz con un valor estático.



Presione para llenar todos los elementos de la matriz con el valor "Venezuela".


isArray()


El método isArray () determina si un objeto es una matriz.



Presione para comprobar si "frutas" es una matriz.


pop()


El método pop () elimina el último elemento de una matriz y devuelve ese elemento.



Presione para eliminar el último elemento de la matriz.


push()


El método push () agrega nuevos elementos al final de una matriz y devuelve la nueva longitud.



Presione para agregar un nuevo elemento a la matriz.
