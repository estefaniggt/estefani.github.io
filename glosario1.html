FUNCIONES MATEMATICAS

abs()


Devuelve el valor absoluto de un número.



Presione para mostrar el valor absoluto de -131.99


acos()


El método acos() devuelve el arcocoseno de un número como un valor entre 0 y PI radianes.



Presione para mostrar el arcocoseno de 0.5


acosh()


El método acosh() devuelve el arcocoseno hiperbólico de un número.



Presione para mostrar el arcocoseno hiperbólico de 3


asin()


El método asin () devuelve el arcoseno de un número como un valor entre -PI / 2 y PI / 2 radianes.



Presione para mostrar el arcoseno de 2


pow()


El método pow () devuelve el valor de x elevado a la potencia de y.



Presione para mostrar la potencia de 4 elevado a la 3
